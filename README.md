# I Sound

```bash
git clone https://github.com/codeoD/I-Sound-Music-Player.git

cd I-Sound-Music-Player

npm i

cd ui

npm i

npm run build

cd ..

npm start

```

## 已实现功能

    - 添加歌单

    - 添加歌曲

    - 播放模式

    - 音量调节
